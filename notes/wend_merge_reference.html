Wend — tower merge / juice reference

Animation + interaction target for the merge mechanic. Built for CC. Placeholder art and stat numbers.

Feel: cartoon and fun, not impactful. No screen shake. The merge result wobbles with a gummy jelly squash-and-stretch that settles; impact is a soft pastel poof, not a hard flash.
  • Input — controller / Steam Deck (primary): move cursor → press action to arm the tower (it lifts) → press a direction to merge that way → auto-disarms. Invalid direction (empty / wrong tier / maxed) = reject nudge.
  • Input — mouse: drag a tower onto an adjacent same-tier tower.
  • Merge rule: same tier only. Two of tier N → one tier N+1. Pure merge, max T10 (2ⁿ cost; high tiers are aspirational).
  • Risk: merging empties the source tile → leaves a hole in the maze. Build-timer pressure.
  • Appearance morph: Barrel count = multishot count and a tier badge shows the exact tier — the functional read lives in barrels/structure, so the body stays a pure skin slot. Tier-investment tell = a ground glow under the tower that intensifies with merge tier (color per board); canonical spec design/TOWER_AURA.md. (The per-tier body-color ramp shown in the demo below is superseded — kept only as a historical preview.)
  • Multishot: ×2 at T3, ×3 at T6, ×4 at T10 (cap 4). Other stats scale per tier — numbers TBD, balance later. DPS = damage × shots × fire rate, so multishot is the dominant lever.
